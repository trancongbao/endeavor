# Lessons

A standard lesson includes:
+ title
+ text with images
+ audio of text
+ translation of text
+ words
+ grammar
+ exercises

A video lesson includes:
+ title
+ video with subtitles
+ translation of subtitles
+ words section
+ grammar section
+ exercise section

Each lesson also has a corresponding spaced-repetition flash-cards decks for reviewing vocabulary and grammar.

See [Decks](./Decks/README.md) for more details.
